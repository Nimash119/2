Для того чтобы на сайте показывать, кто оставил комментарий, а также количество онлайн пользователей, тебе нужно будет использовать JavaScript (или серверную часть на Python, Node.js и т.д.) для отслеживания активности.

Вот пример реализации с использованием JavaScript и HTML. Будем считать, что пользователи могут оставить комментарий, и их активность отслеживается через онлайн-статус.

  1. HTML: Это основная разметка для комментариев и отображения информации о том, кто оставил комментарий.

  2. JavaScript: Отслеживает количество онлайн пользователей и позволяет отображать имя пользователя, который оставил комментарий.

Пример кода

HTML + CSS (Frontend):

html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Комментарии и Онлайн пользователи</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 20px;
        }

        .comment-section {
            margin-top: 30px;
        }

        .comment {
            border: 1px solid #ccc;
            padding: 10px;
            margin-bottom: 10px;
            border-radius: 5px;
        }

        .commenter {
            font-weight: bold;
            color: #4CAF50;
        }

        .online-users {
            margin-top: 20px;
            font-size: 18px;
        }

        .online-users span {
            font-weight: bold;
            color: #ff6347;
        }
    </style>
</head>
<body>
    <h1>Комментарии на сайте</h1>

    <!-- Форма для добавления комментария -->
    <div class="comment-section">
        <h2>Оставьте комментарий</h2>
        <input type="text" id="username" placeholder="Ваше имя" />
        <textarea id="comment" placeholder="Ваш комментарий..." rows="4" style="width: 100%;"></textarea>
        <button onclick="submitComment()">Отправить</button>
    </div>

    <div id="commentsContainer"></div>

    <!-- Онлайн пользователи -->
    <div class="online-users">
        <h3>Онлайн пользователи:</h3>
        <span id="onlineCount">0</span> человек(а) онлайн
    </div>

    <script>
        // Массив для хранения комментариев
        const comments = [];

        // Счетчик онлайн пользователей
        let onlineCount = 0;

        // Отображаем комментарии
        function displayComments() {
            const commentsContainer = document.getElementById('commentsContainer');
            commentsContainer.innerHTML = ''; // очищаем контейнер перед рендером

            comments.forEach(comment => {
                const commentDiv = document.createElement('div');
                commentDiv.className = 'comment';
                commentDiv.innerHTML = `
                    <div class="commenter">${comment.username}</div>
                    <div>${comment.text}</div>
                `;
                commentsContainer.appendChild(commentDiv);
            });
        }

        // Добавление комментария
        function submitComment() {
            const username = document.getElementById('username').value;
            const commentText = document.getElementById('comment').value;

            if (username && commentText) {
                comments.push({ username, text: commentText });
                displayComments();
                clearInputs();
            } else {
                alert('Пожалуйста, заполните все поля!');
            }
        }

        // Очищаем поля ввода после отправки комментария
        function clearInputs() {
            document.getElementById('username').value = '';
            document.getElementById('comment').value = '';
        }

        // Функция для отслеживания онлайн пользователей
        function updateOnlineCount() {
            // Для примера, давай будем случайно увеличивать/уменьшать онлайн пользователей
            onlineCount = Math.floor(Math.random() * 100) + 1;
            document.getElementById('onlineCount').textContent = onlineCount;
        }

        // Обновляем онлайн пользователей каждую секунду
        setInterval(updateOnlineCount, 1000);
    </script>
</body>
</html>

Объяснение кода:

  1. HTML:

    • У нас есть простая форма для ввода имени и комментария.

    • Выводим все комментарии в блоке #commentsContainer.

    • Отображаем количество онлайн пользователей в блоке #onlineCount.

  2. JavaScript:

    • Функция submitComment() добавляет новый комментарий в массив comments и обновляет отображение комментариев.

    • Функция updateOnlineCount() случайным образом генерирует число онлайн пользователей и обновляет отображение на странице каждую секунду.

Как это работает:

  • Когда пользователь оставляет комментарий, он появляется на странице в соответствующем блоке.

  • Количество онлайн пользователей обновляется каждую секунду с помощью функции setInterval, которая случайным образом генерирует число онлайн пользователей.

Если хочешь, чтобы этот функционал работал с сервером, тебе нужно будет подключить серверную часть (например, на Node.js или Python) для реального отслеживания пользователей и сохранения комментариев в базе данных.
